# Chrome Web Store Assets

Place the following files in this directory before submitting to the Chrome Web Store:

## Required
- **icon-128.png** — 128x128 extension icon (high quality, branded)
- **screenshot-1.png** — 1280x800 or 640x400 screenshot of the popup
- **screenshot-2.png** — Screenshot of the notes editor in action
- **screenshot-3.png** — Screenshot of the options page (connected accounts)

## Recommended
- **promo-small.png** — 440x280 promotional tile
- **promo-marquee.png** — 1400x560 marquee image

## Store Listing Text
- **Short description** (132 chars max): Take meeting notes and auto-save them to Google Drive, OneDrive, or Dropbox.
- **Detailed description**: See README.md features section.
- **Category**: Productivity
- **Language**: English
